Architecture Implémentée

Client System Flow:
1. Dashboard (client.dashboard) - Vue principale
   ├─ Active Orders Widget
   ├─ Quick Actions (Menu, Cart, Orders, Invoices)
   └─ Statistics

2. Menu (client.menu) - Navigation
   ├─ Category Filter
   ├─ Search
   └─ Add to Cart (AJAX)

3. Cart (client.view-cart) - Review
   ├─ Item Management (qty update/remove)
   ├─ Summary (HT, TVA, TTC)
   └─ Checkout

4. Checkout (client.checkout-form) - Configuration
   ├─ Order Type Selection (sur_place/a_emporter/livraison)
   ├─ Table Selection (sur_place only)
   ├─ Delivery Address (livraison only)
   └─ Submit

5. Order Detail (client.order-detail) - View
   ├─ Order Info
   ├─ Timeline
   ├─ Items
   ├─ Payment (if needed)
   └─ Actions (print, cancel, pay)

6. Order History (client.order-history) - List
   └─ Paginated Commands

7. Invoices (client.invoices) - Factures
   └─ Paginated Invoices
